# Time Tracking API

Backend API for managing users, authentication, and tracked work time.

## Planned Features

- User accounts with admin control
- User profiles with basic personal info
- Time logs for daily work hours
- Login and register endpoints

## Development

```bash
npm run start:dev
npm run test
npm run test:e2e
npm run test:cov
npm run format
```

## Production

```bash
npm install
npm run build
npm run start:prod
```
